# OquSpace - Онлайн оқыту платформасы

## Жалпы шолу

OquSpace - қазақ тілінде білім беру курстарын ұсынатын React-те жасалған заманауи онлайн оқыту платформасы. Платформа пайдаланушыларға курстарды шолуға, сабақтарға қол жеткізуге, тесттер орындауға және оқу барысын бақылауға мүмкіндік беретін интуитивті интерфейсті ұсынады.

## Мүмкіндіктер

- **Пайдаланушы аутентификациясы**: Қауіпсіз тіркеу және кіру функционалдығы
- **Курстар каталогы**: Қолжетімді курстарды шолу және іздеу
- **Сабақ көрсеткіші**: Әр түрлі форматтағы сабақ мазмұнына қол жеткізу
- **Тест жүйесі**: Білімді тексеруге арналған интерактивті тесттер
- **Профиль басқару**: Пайдаланушы профилі мен оқу барысын көру және жаңарту
- **AI чат көмекшісі**: Оқу барысында көмек пен нұсқаулық алу
- **Тесттер орындау**: Тесттер орындау және нәтижелерін көру
- **Икемді дизайн**: Кез келген құрылғыдан (десктоп, планшет, мобильді) қол жеткізу

## Технологиялар стеги

- **Frontend**: React (v19.0.0)
- **Маршруттау**: React Router (v7.0.2)
- **Стильдер**: Styled Components (v6.1.13)
- **Күй басқару**: React хуктары жергілікті күй үшін
- **API интеграциясы**: Арнайы fetch негізіндегі сервис қабаты
- **Тестілеу**: React Testing Library

## Жұмысты бастау

### Алғышарттар

- Node.js (v14 немесе жоғары)
- npm (v6 немесе жоғары)

### Орнату

1. Репозиторийді клондаңыз
2. Тәуелділіктерді орнатыңыз:

```
npm install
```

### Қолданбаны іске қосу

1. Даму серверін іске қосыңыз:

```
npm start
```

2. Браузерде [http://localhost:3000](http://localhost:3000) ашыңыз

## Негізгі компоненттер

### Беттер

- **MainPage**: Курстар каталогы мен ерекшеленген мазмұнды көрсететін басты бет
- **LoginPage/RegisterPage**: Пайдаланушы аутентификациясы
- **ProfilePage**: Пайдаланушы профилін басқару
- **CoursePage**: Нақты курс туралы мәліметтер
- **LessonPage**: Сабақ мазмұнын көру
- **AboutPage**: Платформа туралы ақпарат

### Компоненттер

- **Header**: Навигация және пайдаланушы мәзірі
- **Quiz**: Интерактивті тест компоненті
- **TestTaker/TestEditor/TestResults**: Тестілеумен байланысты компоненттер
- **AIChatButton**: Оқуды қолдау үшін AI көмекшісі

## API интеграциясы

Қолданба бэкенд API-мен (әдепкі бойынша http://127.0.0.1:8000/api мекенжайында жұмыс істейтін) арнайы сервис қабаты арқылы байланысады. Негізгі сервистерге мыналар кіреді:

- **authService**: Пайдаланушы аутентификациясын басқарады
- **courseService**: Курстармен байланысты операцияларды басқарады
- **testService**: Тесттер мен тестілеу функционалдығын басқарады

## Аутентификация

Аутентификация session storage-де сақталатын JWT токендерін пайдаланып іске асырылады. Аутентификация ағынына мыналар кіреді:

- Кіру/тіркелу
- Токенді басқару
- Қорғалған маршруттар

# OquSpace - Техникалық құжаттама

## Архитектураға шолу

OquSpace - RESTful бэкенд API-мен байланысатын React-те жасалған бір беттік қолданба (SPA). Қолданба міндеттерді анық бөлумен компонентке негізделген архитектураны қолданады.

## Frontend архитектурасы

### Негізгі құрылым

```
frontend/
├── public/            # Статикалық файлдар
├── src/
│   ├── components/    # Қайта пайдаланылатын UI компоненттері
│   ├── pages/         # Қолданба беттері/маршруттары
│   ├── services/      # API және бизнес логикасы
│   ├── hooks/         # Арнайы React хуктары
│   ├── App.js         # Маршруттауы бар негізгі компонент
│   └── index.js       # Қолданба кіру нүктесі
└── package.json       # Тәуелділіктер мен скрипттер
```

### Күй басқару

Қолданба React-тің кіріктірілген күй басқару шешімдерін пайдаланады:

- **useState**: Компоненттің жергілікті күйі үшін
- **useEffect**: Сыртқы әсерлер мен деректерді алу үшін
- **useContext** (жасырын түрде): Аутентификация күйі мен тақырыбы үшін

### Маршруттау

Маршруттау React Router v7 көмегімен жүзеге асырылады:

```jsx
<Router>
  <Routes>
    <Route path="/" element={<MainPage />} />
    <Route path="/login" element={<LoginPage />} />
    <Route path="/register" element={<RegisterPage />} />
    <Route path="/profile" element={<ProfilePage />} />
    <Route path="/course/:id" element={<CoursePage />} />
    <Route path="/course/:courseId/lesson/:lessonId" element={<LessonPage />} />
    <Route path="/about" element={<AboutPage />} />
  </Routes>
</Router>
```

## API интеграциясы

### Сервис қабаты

API байланысы сервис қабаты үлгісі арқылы өңделеді:

1. **api.js**: Сұрау/жауап өңдеумен негізгі API функционалдығы
2. **authService.js**: Аутентификация операциялары
3. **courseService.js**: Курстармен байланысты операциялар
4. **testService.js**: Тесттер мен тестілеу функционалдығы

### API Эндпоинттері

#### Аутентификация

- `POST /api/auth/register`: Пайдаланушы тіркелуі
- `POST /api/auth/login`: Пайдаланушы кіруі
- `POST /api/auth/refresh`: Қатынау токенін жаңарту
- `POST /api/auth/logout`: Пайдаланушы шығуы

#### Курстар

- `GET /api/courses`: Барлық курстарды алу
- `GET /api/courses/{id}`: Курс мәліметтерін алу
- `GET /api/courses/{courseId}/lessons/{lessonId}`: Сабақ мәліметтерін алу
- `POST /api/courses/{courseId}/enroll`: Курсқа жазылу
- `POST /api/courses/{courseId}/progress`: Курс барысын жаңарту

#### Пайдаланушы профилі

- `GET /api/users/profile`: Пайдаланушы профилін алу
- `PUT /api/users/profile`: Пайдаланушы профилін жаңарту
- `GET /api/users/progress`: Оқу барысын алу

#### Тесттер мен сынақтар

- `GET /api/tests/{id}`: Тест мәліметтерін алу
- `POST /api/tests/{id}/submit`: Тест жауаптарын жіберу
- `GET /api/tests/{id}/results`: Тест нәтижелерін алу

## Негізгі компоненттер

### Header компоненті

Header компоненті навигация мен пайдаланушы аутентификация күйін ұсынады:

```jsx

```

Негізгі мүмкіндіктер:

AI көмегімен оқуды қолдауды ұсынады:

```jsx
<AIChatButton courseId={courseId} lessonId={lessonId} />
```

## Аутентификация ағыны

1. **Кіру/Тіркелу**:

   - Пайдаланушы тіркелгі деректерін жібереді
   - Сервер тексереді және JWT токенін қайтарады
   - Токен sessionStorage-де сақталады

2. **Сұрау аутентификациясы**:

   - Аутентификация токені API сұрауларына қосылады
   - Рұқсат берілмеген жауаптар кіру бетіне бағыттайды

3. **Токенді басқару**:
   - Токен мерзімінің аяқталуын өңдеу
   - Автоматты жаңарту механизмі

## Жұмысты бастау

### Аккаунт құру

1. OquSpace сайтына кіріңіз [your-website-url.com](https://your-website-url.com)
2. Жоғарғы оң жақ бұрыштағы "Тіркелу" түймесін басыңыз
3. Тіркеу формасын деректеріңізбен толтырыңыз:
   - Толық аты-жөні
   - Электрондық пошта мекенжайы
   - Құпия сөз (кемінде 8 таңба)
4. Пайдалану шарттарымен келісіңіз
5. "Аккаунт құру" түймесін басыңыз
6. Поштаңызға жіберілген сілтемені басу арқылы электрондық поштаңызды растаңыз

### Жүйеге кіру

1. Жоғарғы оң жақ бұрыштағы "Кіру" түймесін басыңыз
2. Электрондық поштаңыз бен құпия сөзді енгізіңіз
3. "Кіру" түймесін басыңыз
4. Құпия сөзіңізді ұмытып қалсаңыз, "Құпия сөзді ұмыттым" түймесін басып, нұсқауларды орындаңыз

### Платформада жылжу

**Негізгі навигация:**

- **Басты бет**: Курстар каталогы бар басты бетке қайтады
- **Менің курстарым**: Сіз жазылған курстарды көрсетеді
- **Профиль**: Профиліңіз бен параметрлерге қол жеткізу
- **Біз туралы**: OquSpace туралы ақпарат

## Платформаны пайдалану

### Курстарды шолу

1. Басты бетте курстар каталогын көресіз
2. Нақты курстарды табу үшін іздеу өрісін пайдаланыңыз
3. Тізім бойынша айналдыру арқылы курстарды шолыңыз
4. Қосымша мәліметтер көру үшін курс картасын басыңыз

### Курсқа жазылу

1. Курстың мәліметтер бетінен "Курсқа жазылу" түймесін басыңыз
2. Жазылуыңызды растаңыз
3. Курс енді "Менің курстарым" бөлімінде пайда болады

### Сабақтарды өту

1. Жазылған курсыңыздан "Оқуды бастау" түймесін басыңыз немесе тоқтаған жерден жалғастырыңыз
2. Бүйірлік панель немесе келесі/алдыңғы түймелерін пайдаланып сабақтар арасында жылжыңыз
3. Аяқтаған сабақтарыңызды "Аяқталды" деп белгілеңіз
4. Прогресс жолағы арқылы ілгерілеуіңізді қадағалаңыз

### Тесттерді орындау

1. Тесттер сабақтардың немесе бөлімдердің соңында пайда болады
2. Әр сұрақты мұқият оқып, жауабыңызды таңдаңыз
3. Жауаптарыңызды тексеру үшін "Жіберу" түймесін басыңыз
4. Нәтижелеріңізді және әр сұрақ бойынша түсіндірмелерді қарап шығыңыз
5. Ұпайыңызды жақсарту үшін тесттерді қайта орындаңыз

### Емтихандарды тапсыру

1. Емтихандар сіздің біліміңізді барынша жан-жақты бағалау болып табылады
2. Олар курс бөлімдерінің немесе курстың соңында пайда болады
3. Әр емтихан үшін берілген нұсқауларды орындаңыз
4. Уақыт бітпей тұрып жауаптарыңызды жіберіңіз (егер қолданылса)
5. Емтихан нәтижелеріңіз бен жақсартуға арналған салаларды қарап шығыңыз

## Профильді басқару

1. Жоғарғы оң жақ бұрыштағы профиль суретіңізді немесе атыңызды басыңыз
2. Ашылмалы мәзірден "Профиль" опциясын таңдаңыз
3. Профиль бетінен келесі әрекеттерді орындай аласыз:
   - Жеке ақпаратыңызды жаңарту
   - Құпия сөзіңізді өзгерту
   - Оқу прогресіңізді көру
   - Сертификаттарыңыз бен жетістіктеріңізді көру
   - Аккаунт параметрлерін басқару

## Жиі қойылатын сұрақтар

### Оқу барысымды қалай қадағалаймын?

Сіздің оқу барысыңыз сабақтар мен тесттерді аяқтаған сайын автоматты түрде бақыланады. Жалпы прогресіңізді профиль бетінде және курсқа тән прогресті әр курс ішінде көре аласыз.

### Курстарға мобильді құрылғылардан қол жеткізе аламын ба?

Иә! OquSpace толықтай икемді және смартфондарда, планшеттерде және компьютерлерде жұмыс істейді.

### Сертификатты қалай аламын?

Сертификаттар барлық қажетті сабақтарды, тесттерді және емтихандарды сәтті аяқтағаннан кейін беріледі. Сертификаттарыңызды профиль бетінен жүктеп алып, бөлісе аласыз.

### Курс туралы пікірімді қалай білдіре аламын?

Әр курстың соңында сізден пікір сұралады. Сонымен қатар, курс мәзіріндегі "Курс пікірі" опциясы арқылы пікір жібере аласыз.

### Техникалық қиындықтар туындаса не істеймін?

Профиль мәзіріндегі "Қолдау қызметіне хабарласу" опциясы арқылы қолдау тобымызға хабарласыңыз немесе жалпы мәселелер бойынша тез арада көмек алу үшін AI Chat Көмекшісін пайдаланыңыз.
